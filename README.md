# QuizApp
Small python script for Windows to beat quizzes using Google Vision.

Pre-requisites:
- Google Vision API set up via Environment Variable.
- Google Search API key.

Packages required:
1. PIL
2. Enum
3. Requests
4. Colorama

### Usage
`ocr.py <image filename>`
